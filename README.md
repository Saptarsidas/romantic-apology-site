# Romantic Apology Website (React + Tailwind)

A 7-page interactive apology website with mini-games, smooth route transitions, and mobile-friendly design.

## Tech

- React (Vite)
- JavaScript
- Tailwind CSS
- React Router
- Framer Motion

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
